How to use this without it dying

1. Capture rough. In the unit, dictate one concept into the keyboard mic, hit Save, move on. Don't sort.
2. Triage weekly. Open Review, filter to Inbox, assign domains in ~5 min.
3. Hunt gaps monthly. Gap Map shows what you haven't touched — pick a red subtopic to read up on.
4. Teach to keep it. When a topic has 3+ concepts, Build Handout and turn it into a Bedside Reference card.
